# Tanks Assembly Game
Two-player competitive tank fighting game in Easy 68k assembly. Developed as a graduate student at Florida Interactive Entertainment Academy in Fall 2022

1. Install [Easy68K](http://www.easy68k.com/)
2. Run main.X68
